Node.js APIs VS Browser APIs

Both allow Javascipt to do powerful things, but they run in different environments.

1. Execution Enviornment
Node.js APIs
Runs on= This will run on server/local machine
Contolled by= This will controlled by Node.js runtime
Access level= This will access full system access.

Browser APIs
Runs on= This will run on web browser only.
Controlled by = Browser (chrome, firefox,safari)
Access level=Limited(for security);


2.Core Purpose
Node.js APIs:- Backend logic , File system access, Server creation, Database handling
Browser APIs:- UI & user interaction, DOM manipulation, Rendering Web Pages, Event handling
